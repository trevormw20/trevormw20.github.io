PARC: Repitition

Humble Bundle

www.humblebundle.com

There can be a lot of information that is needed on this website for the fewer to find what they are interested in. Because this website is not something someone goes to find a specific item, rather browse to see if anything is interesting. With the same elements found on each targeted item. The viewer is easiyly able to find how many days left, what it is, how much it costs and what is the value of the bundle. The repeated elements make it so that if the viewer wants to find something specific about each bundle. They know where to look.

PARC Contrast

Blender

www.blender.org

The website has a lot of white space which creates a clean space. When something out of the ordinary pops up like the download button. Your eye quickly jumps to the contrasting element.

Visual Hierarchy

RoadToVR

www.roadtovr.com

Visual Hierarchy is shown here with the lead article being the biggest and easiest to see what the image is about. This makes it easy for the viewer to get attached and it is easy to click on compared to the other articles shown on the home page.
